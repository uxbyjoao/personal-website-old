This is my own personal website, created using **TailwindCSS** and **Next.js**.

## Getting started

---

1. Install dependencies:

```
npm install
```

2. Start development server:

```
npm run dev
```

3. Build Next.js site locally:

```
npm run build
```

4. Preview local build (run `npm run build` first):

```
npm run serve
```
